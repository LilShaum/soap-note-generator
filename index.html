How to use the SOAP Generator
1
Pick a note type
Click any note type on the left side panel. It will turn green when selected.
2
Fill in the form
Use the dropdowns and text fields to enter the patient's information. For checkboxes in the S section, tick any symptoms the patient has. For checkboxes in the O section, tick only if something abnormal was found — leave them all blank if the exam was normal.
3
Generate the note
Click the green Generate Note button at the bottom of the form. The note will appear below it.
4
Copy into Med Access
Click the Copy button in the dark bar above the note. Then paste it into the appropriate field in Med Access using Ctrl + V.
5
Start a new note
Click the Clear button to reset the form, or simply click a different note type on the left to switch.
C
Compact Mode
Click Compact Mode in the top bar to switch to shorter notes. The same form is used — only the output changes. Click it again to go back to the full note.
